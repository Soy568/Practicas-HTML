Css es el lenguaje utilizado para diseñar un documento Html, describe como mostrar los elementos
Cascading Style Sheets puede controlar el diseño de varias paginas web sin interferir los estilos de otras paginas web.

Sintaxis Css

Selectores

  1. Selectores simples (seleccione elementos según el nombre, la identificación, la clase)
  2. Selectores del combinador (seleccione elementos basados en una relación específica entre ellos)
  3. Selectores de pseudoclase (seleccionar elementos en función de un estado determinado)
  4. Selectores de pseudoelementos (select y aplicar estilo a una parte de un elemento)
  5. Selectores de atributos (seleccionar elementos en función de un atributo o valor de atributo)
