Eventos

Los eventos son cosas que le suceden a los elementos, y JavaScript reacciona a estos.
JavaScript permite ejecutar codigo al detectar un evento.
HTML permite agregar atributos de control de eventos.

Eventos comunes DOM Events 85

  • onclick : se hace clic en un elemento.
  • onchange : se ha cambiado un elemento.
  • onmouseover : se mueve el mouse sobre un elemento.
  • onmouseout : se aleja el mouse de un elemento.
  • onkeydown : se presiona una tecla.
  • onload : el navegador a terminado de cargar la pagina.
